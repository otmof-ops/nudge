# Code of Conduct

**OFFTRACKMEDIA Studios** — ABN 84 290 819 896

## Our Standard

This project is maintained by OFFTRACKMEDIA Studios. We expect all participants — contributors, issue reporters, and community members — to conduct themselves professionally.

## Expected Behavior

- Be respectful and constructive in all interactions
- Provide clear, actionable feedback in issues and pull requests
- Accept constructive criticism gracefully
- Focus on what is best for the project and its users
- Respect differing viewpoints and experiences

## Unacceptable Behavior

- Harassment, insults, or derogatory comments
- Personal or political attacks
- Publishing others' private information without consent
- Trolling, spam, or deliberately derailing discussions
- Any conduct that would be considered inappropriate in a professional setting

## Enforcement

Instances of unacceptable behavior may be reported to `otmof.mail@gmail.com`. All reports will be reviewed and investigated. The project maintainers reserve the right to remove, edit, or reject comments, commits, issues, and pull requests, and to temporarily or permanently ban any contributor for behavior they deem inappropriate, threatening, or harmful.

## Scope

This Code of Conduct applies to all project spaces — GitHub issues, pull requests, discussions, and any other communication channels associated with this project.
